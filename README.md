# ts-cli-starter

A minimal starter project for typescript cli tools

## Development

Clone the project

```bash
  git clone git@github.com:smnhgn/ts-cli-starter.git
```

Go to the project folder

```bash
  cd ts-cli-starter
```

Install dependencies

```bash
  npm install
```

Symlink the project to the global npm folder

```bash
  npm link
```

Run the project

```bash
  npm run dev
```

Run the cli tool

```bash
  ts-cli-starter
```
